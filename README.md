# PlantUML macro in LibreOffice

## Compile

Just run ./make.sh Libo_PlantUML (on linux)

## Test

Tested on 

* Ubuntu 16.10 64bit
* Windows 7 64bit

Please contact me if you can confirm that it runs on Mac as well.
